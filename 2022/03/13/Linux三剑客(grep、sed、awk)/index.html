Linux三剑客(grep、sed、awk)

三剑客的功能非常强大，但我们只需要掌握他们分别擅长的领域即可：grep擅长查找功能，sed擅长取行和替换。awk擅长取列

正则表达式

grep

匹配单个字符的元字符

. 任意单个字符

[akt] 任意其中一个字符

- 连续的字符
[a-z] 任意单个小写字母
[A-Z] [a-zA-Z]
[0-9] [a-zA-Z0-9]
^ 取反

[^a-z]

[[:space:]] 任意单个空白字符

匹配字符出现的次数

* 前一个字符出现任意次
ab*
.*
? 前一个字符最多出现一次 可有可无
ab? a ab
+ 前一个字符至少出现一次
ab+ ab abb abbb
{3} 前一个字符精确出现3次
ab{3} abbb
{2, 5} 至少2次，最多5次
{2,} 至少出现2次

分组 \( \)

\(ab\)+

匹配字符出现位置

^string 以string开头
string$ 以string结尾
^$ 空行

sed

工作原理

将文件逐行读入到模式空间(内存)，在内存中对文件进行修改，修改完毕后，默认会把模式空间所有内容显示到屏幕上
逐行处理
默认情况下，不会修改原文件

sed命令格式

line常用的写法

line 哪些行 CMD 操作
line不写，表示对文件每一行进行操作

行号 10
起始行号,终止行号
5,10
5,+3
/正则表达式/ /^#/
/正则表达式/,/正则表达式/

常用操作

d 删除整行
p 显示整行
a \内容 追加内容
i \内容 插入内容

​

c \内容 整行替换
w 文件名称 另存为
r 文件名称 合并文件
= 显示行号/统计行数
n 读取下一行
s/旧内容/新内容/[修饰符]

​ 文件查找替换
​ 旧内容支持正则表达式

反向引用

​ \1 \2 \3
​ 依次表示引用正则表达式中第一个分组内容、第二个分组内容、依次类推

&: 匹配的所有旧内容

常用选项

-n 取消显示模式空间的内容
-i 修改原文件
-e 同时做多个修改
-f 操作文件
-r 支持扩展正则表达式
–follow-symlinks 软链接文件

awk

默认情况下，awk可以使用空白字符分割文本, awk内部可以分别**$1, $2**变量代表第一段内容
-F选项指定分割符
逐行处理

awk命令格式

常用action

print操作

printf操作
格式化输出内容
使用格式
printf “格式格式”,内容,内容
占位符
%s 字符串
%d 整数
%f 浮点数

设置字符的宽度
%10s

默认数据在指定的宽度字符是以右对齐的方式显示

设置左对齐
%-10s

默认没有换行，如果需要换行，手动添加换行符\n

awk常用内置变量

FS 保存行分割符，默认空白

相当于-F选项的作用

OFS 指定awk显示多段内容时，多段内容间的分隔符
NR 记录行号

处理多个文件时，所有文件的行号是连续记录的

FNR 记录行号

处理多个文件时，所有文件的行号是独立记录的

NF 记录awk分割行后的段数
FILENAME 保存awk正在处理的文件名

awk自定义变量

-v 变量名称=值

awk常用文本处理模式

表达式

比较运算符
==, >, >=, <, <=, !=
数据 ~ 正则表达式
判断数据是否可以被正则表达式
数据 !~ 正则表达式
逻辑运算符
&& 并且
|| 或者

/正则表达式/
/正则表达式/,/正则表达式/
BEGIN{}

定义awk开始处理内容前执行的操作

END{}

定义awk处理内容后要执行的操作

awk条件判断

1、语法
if (条件) {操作}
if (条件) {操作} else {操作}
if (条件) {操作} else if (条件) {操作}

[root@localhost ~]# awk -F: ‘{if($3==0) {print $1, “是管理员”} else {print $1, “是普通用户”}}’ /etc/passwd
[root@localhost ~]# awk -F: -v bash_number=0 -v nologin_number=0 ‘{if($7==”/bin/bash”) {bash_number++} else if($7==”/sbin/nologin”) {nologin_number++}}END{print “bash用户数”, bash_number, “nologin用户数”, nologin_number}’ /etc/passwd

awk循环

for循环语法

awk数组

定义数组
获取数组中所有数据

awk的内置函数

split(字符串,数组,行分割符)

使用分割符分割字符串, 将分割后的字符串保存到指定数组

length(string)

返回string字符串的字符个数

substr(string,start [,length])

取string字符串中的子串，从start开始，取length个；start从1开始计数

system(command)

支持在awk内部调用shell命令

systime()

返回系统当前时间

tolower(string)

将string中的所有字母转为小写

toupper(string)

将string中所有字母转换为大写
